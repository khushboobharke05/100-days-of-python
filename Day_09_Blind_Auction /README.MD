# Blind Auction Program (Python)

This is a simple Python project that simulates a blind auction system.

## Features
- Takes user input for name and bid amount
- Stores all bids in a dictionary
- Allows multiple users to place bids
- Determines the highest bidder
- Displays the winner with the highest bid

## How it works
1. Users enter their name and bid
2. Data is stored in a dictionary
3. Program continues until no more bidders
4. The highest bid is calculated using a loop
5. Winner is displayed

## Concepts Used
- Functions
- Dictionaries
- Loops (while, for)
- Conditional statements (if-else)

## Example
Input:
A → 100  
B → 300  
C → 200  

Output:
The winner is B with a bid of $300
